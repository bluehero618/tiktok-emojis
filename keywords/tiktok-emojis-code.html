Using TikTok Emoji Codes

Our TikTok emojis code reference provides Unicode and HTML codes for all TikTok emojis. You can use these codes in your web projects, applications, or any platform that supports emoji rendering.

To use an emoji in HTML, you can use its HTML entity (e.g., &#128512; for 😀) or directly insert the emoji character in UTF-8 encoded documents.

For JavaScript applications, you can use the unicode escape sequence (e.g., \uD83D\uDE00 for 😀) or directly insert the emoji character in your source code if it's saved with UTF-8 encoding.
