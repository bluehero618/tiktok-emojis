How to Add TikTok Emojis to Discord

Step 1: Download TikTok Emojis

Browse our collection and download the PNG versions of your favorite TikTok emojis. All our emojis are optimized for Discord with transparent backgrounds and appropriate sizes.

Step 2: Access Discord Server Settings

Open Discord and navigate to your server where you want to add the emojis. Click on the server name to open the dropdown menu, then select "Server Settings" and go to the "Emoji" tab.

Step 3: Upload Your TikTok Emojis

Click the "Upload Emoji" button and select the TikTok emoji PNG files you downloaded. Give each emoji a name that will be used when typing the emoji command (e.g., :tiktok_smile:).

Step 4: Use Your New TikTok Emojis

Once uploaded, you can use your TikTok emojis in any channel on your server. Type ":" followed by the name you gave the emoji to bring up the emoji picker, or click the emoji button in the chat box.

Discord Emoji Tips

Server Emoji Limits

Standard Discord servers can have up to 50 custom emojis. Servers with Boost Level 1 get 100 emoji slots, Level 2 gets 150 slots, and Level 3 gets 250 slots.

Emoji Size Requirements

Discord accepts images up to 256KB in size. For best quality, use PNG files with dimensions of 128x128 pixels or larger with a transparent background.

Naming Conventions

Use descriptive names for your emojis to make them easy to find. Names can only contain alphanumeric characters and underscores, and must be at least 2 characters long.

Emoji Permissions

Server owners can control which roles have permission to use certain emojis. This can be managed in the server settings under "Roles" and then "Permissions."
