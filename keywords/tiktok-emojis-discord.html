Discord Emoji Tips

Server Emoji Limits

Standard Discord servers can have up to 50 custom emojis. Servers with Boost Level 1 get 100 emoji slots, Level 2 gets 150 slots, and Level 3 gets 250 slots.

Emoji Size Requirements

Discord accepts images up to 256KB in size. For best quality, use PNG files with dimensions of 128x128 pixels or larger with a transparent background.

Naming Conventions

Use descriptive names for your emojis to make them easy to find. Names can only contain alphanumeric characters and underscores, and must be at least 2 characters long.

Emoji Permissions

Server owners can control which roles have permission to use certain emojis. This can be managed in the server settings under "Roles" and then "Permissions."
