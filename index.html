How to use TikTok emojis?

Click the copy icon on any emoji card to copy it to your clipboard. Then you can paste it in TikTok comments, bio, or anywhere else that supports emojis.

Click the download icon on any emoji card to download a high-quality PNG image of the emoji. These TikTok emoji PNGs can be used in your designs, documents, or anywhere you need a visual emoji.

Can I use TikTok emojis outside of TikTok?

Yes! You can copy any TikTok emoji and use it in other social media platforms, messaging apps, or anywhere that supports Unicode emojis. For special TikTok emojis, you may need to download and use the PNG images.

How to get TikTok emojis on keyboard?

TikTok emojis are standard Unicode emojis that are already available on most modern keyboards. If you're looking for specific TikTok-themed stickers, those are only available within the TikTok app. Some keyboard apps like Facemoji offer TikTok-style emojis as well.

What are trending TikTok emojis?

Trending TikTok emojis are those that are currently popular in TikTok comments and videos. You can find them in our "Trending" category.

Where can I find TikTok emoji codes?

Each emoji on our site includes its Unicode code point (e.g., U+1F600). This is the universal code that identifies each emoji across different platforms. You can also see the emoji's name which can be used in some TikTok comment features.
